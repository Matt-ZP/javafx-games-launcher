# JavaFX Games Launcher

A desktop games launcher built using Java and JavaFX as part of my software development learning journey.

## Features

### Launcher
- Scene-based navigation
- Main menu system
- Back to Menu functionality
- Exit application button

### Guessing Game
- Random number generation (1–50)
- Attempt tracking
- Invalid input tracking
- Input validation
- Enter key support
- Play Again functionality
- Separation of UI and game logic

## Planned Games
- Rock Paper Scissors
- Hangman
- Tic Tac Toe
